# Change: 修复编辑器UI问题

## Why

当前编辑器界面存在多个影响用户体验的问题：
1. PlainTextEditor 中框选时能看到两层文字（高亮层和输入层），造成视觉混乱
2. 变量交互功能失效，用户无法正常选择和替换变量
3. NotesEditor 覆盖在编辑框内，遮挡文本内容，且位置和功能不符合预期

## What Changes

1. **修复 PlainTextEditor 双层文字问题**：确保高亮层不可见原始文本，只显示渲染后的变量高亮
2. **修复变量交互功能**：恢复 PlainTextEditor 中变量的点击选择和替换功能
3. **重构 NotesEditor 布局**：
   - 将注释板块从编辑框内移出，固定在编辑栏下方
   - 注释板块始终可见，默认占用合适的高度
   - 实现展开按钮功能：在新窗口/独立视图中展示注释内容
   - 添加多语言支持（注释标题、保存按钮等文本）
   - 修复工具栏按钮问题：加粗、斜体导致文本清空，图片、视频功能未生效

## Impact

- **受影响的组件**：
  - `PlainTextEditor.jsx` - 修复双层文字和变量交互
  - `NotesEditor.jsx` - 重构布局和功能
  - `NotesToolbar.jsx` - 修复按钮功能
  - `App.jsx` - 调整布局结构
  - `translations.js` - 添加注释相关翻译键
- **用户体验**：显著改善编辑体验，消除视觉干扰和功能缺陷
- **测试**：需要更新 E2E 测试以验证新的布局和功能
